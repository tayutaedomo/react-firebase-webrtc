# react-firebase-webtrc

Try WebRTC with React and Firebase.

## Setup

```
$ git clone git@github.com:tayutaedomo/react-firebase-webrtc.git
$ cd react-firebase-webrtc
```

### without Docker

```
$ yarn install
$ yarn start
```

### with Docker

```
$ touch .bash_history
$ docker-compose build
$ docker-compose run --rm app yarn install
$ docker-compose up
```

## Available Scripts

```
$ yarn start
$ yarn test
$ yarn build
$ yarn eject
```
